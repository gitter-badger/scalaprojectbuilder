About

Please be aware, this can best be described as a 'beta' project. I am using this as an excuse to learn more about the underlying technologies as well as scratching a personal itch. If you have comments/suggestions etc please email me sbt@@fieldmark.co.uk

This site is built to help you to kickstart a project. You can use this site to generate a template project configured 'ready to go' sbt project with a number of pre-configured features. The Scala ecosystem is amazing rich, but getting a project bootstrapped (even from the g8 templates) can be time consuming and painful.

What it will do

  • Outline a project and download a 'ready to go' project with various features:
    • Play
    • Akka
    • Logging
    • IDE configuration (eclipse/intelliJ)
  • Help to select 'working' configurations of compatible features.
  • Create a project that can immediately be built.packaged and deployed
  • For Play Provide a Heroku deployable project
  • Provide example code to work from
  • Provide (where possible) as many links to the underlying documentation as possible (sbt,akka,play)

What it will not do

  • Build an entire application (it is not a 'site builder')
  • Allow you to 'edit' a previously built and update project
